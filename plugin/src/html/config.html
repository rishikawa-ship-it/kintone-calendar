フィルタ

表示するレコードの絞り込みは kintone 一覧の標準フィルタ機能 をご利用ください。 レコード一覧画面の絞り込み設定がカレンダー表示にも反映されます。

終日フィールド (CHECK_BOX) の選択肢値を入力します。 終日フィールドが未設定の場合は無効です。

カレンダー用ビュー管理

カレンダー表示用のカスタマイズビューを自動で作成・更新します。 手動で <div id="kc-root"></div> を貼り付ける必要はありません。

既定値: カレンダー

セットアップガイド

カレンダーを表示するには、kintone のカスタマイズビューに以下の設定が必要です。

  1. kintone アプリの設定画面を開き、「一覧」タブを選択します。
  2. 「カスタマイズビュー」を追加（または既存のビューを編集）します。
  3. 「HTML」欄に以下のコードを貼り付けます:
    <div id="kc-root" class="kc-root"></div>
  4. アプリを保存・公開します。
  5. カスタマイズビューにアクセスするとカレンダーが表示されます。
注意: このHTML設置はAPIで自動化できないため、管理者が手動で行う必要があります。
